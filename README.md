# vqe-transverse-field-ising
**Transverse Field Ising Model: VQE Benchmarking and Noisy Simulation**
This repository contains Qiskit implementations of the Variational Quantum Eigensolver (VQE) for the frustrated Transverse Field Ising Model (TFIM) on a 3-qubit triangular lattice.

Transverse field Ising Hamiltonian with open boundary conditions:
H = J (Z0 Z1 + Z1 Z2 + Z2 Z0) + h (X0 + X1 + X2) where 
J > 0: ZZ coupling strength
h: transverse field strength
